Gameboy Proyect

Es una Gameboy con botones funcionales y se puede utilizar con teclado o con los propios botones.

Tiene muchas funciones interactuables como poder jugar un combate pokemon, iniciar sesión, etc.

IA Deteccion Personas

Una App con inteligencia artificial que puedes seleccionar o hacer una foto y reconoce si hay o no personas.

Puedes elegir varios modos de ver la prediccion como ver solo el contorno, el rectangulo o todo

Rick Y Morty API

Una App de información de los personajes de la serie Rick Y Morty.

Es una app sencilla utilizando una API opensource de Rick Y Morty.
